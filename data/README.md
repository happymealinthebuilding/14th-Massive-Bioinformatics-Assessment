# Data Directory

This directory should contain the input FASTQ sequencing file.

Example:

VREC1013.fastq.gz

The dataset used in this project can be downloaded from:

https://figshare.com/articles/dataset/ONT_FASTQ_files/7554293?file=14039333

The raw file (~1.72 GB) is not included in this repository due to GitHub file size limits.
